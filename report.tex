\textbf{Цель:} поразмышлять о своих целях и задачах на текущий семестр.

\subsection*{Ход выполнения}

В файле были созданы заголовки «Мой опыт» и «Мои цели» с использованием языка разметки Markdown.

В разделе «Мой опыт» был написан небольшой текст о моем опыте разработки программного обеспечения и добавлена работающая ссылка.

Также были созданы списки целей и задач, которые планируется выполнить в рамках курса.

\section*{Задание 2. Использование Git из командной строки}

\textbf{Цель:} научиться использовать Git из командной строки.

\subsection*{Ход выполнения}

Сначала была проверена версия Git:

git --version

Затем репозиторий был клонирован на локальный компьютер:

git clone https://github.com/NinaGizhe/LB0.git

После клонирования был выполнен переход в папку репозитория:

cd LB0

Далее был создан файл info.md и записаны данные:

echo Gizhe Nina Aleksandrovna > info.md

echo Windows 10 >> info.md

echo 2026-20-01 16:00 >> info.md

Файл был добавлен в Git:

git add info.md

После этого создан коммит:

git commit -m "Добавлен файл info.md"

Была выполнена настройка пользователя Git:

git config --global user.name "NinaGizhe"

git config --global user.email "ninagizhe33@gmail.com"

Проверка настроек:

git config --global --list

Затем изменения были отправлены в GitHub:

git push origin main

\subsection*{Результат}

Файл \textbf{info.md} успешно добавлен в репозиторий.

Содержимое файла:

Gizhe Nina Aleksandrovna

Windows 10

2026-20-01 16:00
